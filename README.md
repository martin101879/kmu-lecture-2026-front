# KMU Capstone - 캡스톤 디자인 개발 실습

계명대학교 컴퓨터공학과 캡스톤 디자인 준비를 위한 강의 사이트입니다.

## 강의 내용

| 주제 | 내용 |
|------|------|
| Spring Boot | 프로젝트 생성, REST API 개발 |
| React | 프로젝트 생성, 컴포넌트 작성 |
| AWS | EC2, S3를 활용한 배포 |
| GitHub | 브랜치 전략, PR 기반 협업 |

## 기술 스택

- React 19 + TypeScript
- Vite
- react-markdown + remark-gfm
- react-syntax-highlighter

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
npm run preview
```
